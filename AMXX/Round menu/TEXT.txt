This menu contains 8 weapons, which an admin can choose from.
When an admin chooses a weapon, it will be activated next round.
When the chosen round finish, everything will return back to normal.

Buy is blocked! Also, a user can only use the weapon of the round that the admin chose.

Currently no pictures are provided, I will try to upload some later.

Please report any bugs, so it can be fixed immediately.

You can change round messages in the source code to whatever you want.

                                // Default value "0"[/code]
