This prints a message in chat for client who connects to the server 
ONLY THE PLAYER WHO CONNECTS SEE IT

Cvars:
connect_message "You Message"
connect_message_enabled "1"


Message format:
!g Green Color
!t Team Color
!n Normal con_color color

%name% Player's name
%ip% Player's IP
%steamid% Steam id

/n New Line


Examples:
!gWelcome To the !nserver

Welcome !g%name% !nto the server!/nYour !tIP !nis !g%ip%/n!nYour !tsteamid !nis !g%steamid%!n!
